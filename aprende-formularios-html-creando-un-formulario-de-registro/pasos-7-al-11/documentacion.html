Proyecto: Formulario de Registro

Para ir dandole estilo a nuestro proyecto, vamos a comenzar agregar algo de CSS. Comienza dándole estilos al elemento <body>. Modificaremos algunas propiedades de este elemento HTML como su altura 1 y ancho 2 asignando valores 3 nuevos a estas propiedades.

Vamos a eliminar el valor por defecto que le da el navegador a la propiedad margin 4. Esta propiedad representa el margen que posee cada elemento, el cual no es más que espacio a su alrededor que lo separa de los demás elementos que lo rodean. Acá hay un ejemplo gráfico de esta propiedad en los elementos HTML.

Otra propiedad que poseen los elementos HTML es background-color 5. A través de esta propiedad se puede definir el color de fondo de los elementos. Adicionalmente, se puede cambiar el color del texto utilizando la propiedad color 6. Para cambiar estas propiedades existen varias unidades, al igual que para los valores del ancho y alto de los elementos.

En este proyecto estaremos creando un formulario, para el cual vamos a necesitar el elemento <form> 7. Básicamente, dentro de este elemento, se anidan los demás elementos necesarios para recibir la información por parte del usuario que está llenando el formulario.

Para organizar los campos del formulario, podemos utilizar un elemento llamado <fieldset> 8. Este permite crear grupos de campos relacionados entre sí dentro del formulario y utiliza etiquetas de apertura y cierre.
