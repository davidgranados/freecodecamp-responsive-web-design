Proyecto: Formulario de Registro

Todo documento HTML debe comenzar con la declaración DOCTYPE 1 para indicar al navegador el tipo de documento y la versión del lenguaje que se está utilizando. Esta declaración se coloca al principio del archivo y se construye de la siguiente manera:

<!DOCTYPE html>

Luego de tener la declaración DOCTYPE, debemos crear un elemento <html> 2 que será el elemento raíz de nuestro código. Este requiere etiquetas de apertura y de cierre.

Los elementos principales que necesitamos dentro de nuestro elemento <html> son el <head> 3 y el <body> 4

Dentro del elemento <head> se coloca información importante para el navegador y los motores de búsqueda. Un elemento que se utiliza dentro del <head> es el <title> 4, este sirve para indicar a la pestaña del navegador y los motores de búsqueda el contenido de nuestra página.

Debajo del elemento <head> colocaremos un elemento <link> 5 el cual permite enlazar hojas de estilo CSS a nuestro HTML. Los atributos6 principales que se declaran en este elemento son rel, que especifica la relación entre el objeto destino y el objeto enlace. Tambien está el href que contiene la URL de un recurso asociado.

En HTML existen los elementos de encabezado7 que son utilizados para indicar el contenido de la página. Estos van del <h1> al <h6> y su importancia dentro del contenido es determinada de manera ascendente.

También existe el elemento <p>8, que sirve para representar párrafos dentro de la página. Por ejemplo, tanto este como el resto de la información que has leído en esta página están contenidos dentro de un elemento <p>.
