Modificando la propiedad margin 1 podemos centrar los elementos dentro de sus contenedores. Y también se puede centrar el texto dentro de estos elementos, modificando el valor de su propiedad text-align.

las propiedades width y height se pueden modificar utilizando diferentes valores o unidades de medida, por ejemplo: podemos utilizar pixeles (px), que es el utilizado comunmente en pantallas y es una unidad de medida absoluta, esto quiere decir que no varían con respecto a otros valores. También se pueden utilizar unidades relativas como viewport height (vh) o viewport width (vw), estas unidades pueden escalar dependiendo del tamaño de una ventana o elemento en el que se encuentre contenido el que está recibiendo los estilos.

Hay elementos a los que por defecto el navegador aplica estilos en sus bordes, estos estilos se pueden sobreescribir o reemplazar cambiando su propiedad border 2. Otra propiedad que tienen los elementos además del margen, es el padding 3. El padding no es más que el espacio que existe entre el contenido de un elemento y sus bordes.

En algunas ocasiones, para poder modificar ciertos elementos en nuestro HTML, necesitamos hacer uso de selectores en CSS conocidos como pseudoclases y pseudoselectores 4. Por ejemplo, una pseudoclase es un selector que marca los elementos que están en un estado específico, como enlaces que han sido visitados o elementos que son el último de su tipo. Los pseudoelementos se comportan de manera similar, sin embargo, actúan como si hubieras añadido un elemento HTML totalmente nuevo en el marcado.

En HTML los elementos pueden mostrarse en línea 5 o en bloque 6. Los elementos inline o en línea, ocupan únicamente el espacio o ancho de su contenido, mientras que los elementos block o en bloque, ocupan todo el ancho disponible en su contenedor. Este comportamiento se puede modificar reemplazando el valor de la propiedad display 7 de dichos elementos.
