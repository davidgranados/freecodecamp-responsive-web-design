#Proyecto Marcadores de Colores con CSS

Pasos atrás aprendiste que el modelo de color [RGB][1] es aditivo. Esto significa que los colores comienzan desde el negro, y cambian a medida que diferentes valores para rojo, azul y verde son agregados.

Una forma sencilla de ver cómo funciona, es a través de la función de CSS `rgb()`. Esta función recibe 3 valores, correspondientes a cada color (rojo, verde, azul) en una escala del 0 (0% del color) a 255 (100% del color) y dependiendo de la combinación de estos valores, crea un nuevo tono de color.

[1]: https://developer.mozilla.org/es/docs/Glossary/RGB
